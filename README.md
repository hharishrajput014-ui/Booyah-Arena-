### *Ab bas 2-3 min wait kar*
Upar hara box aa jayega: `Your site is published at https://hharishrajput014-a.github.io/Booyah-Arena-/`

### *Fir game khol:*
1. Wahi link copy kar le
2. Chrome me paste kar ke Go kar de
3. Tera BOOYAH Arena khul jayega 💪

### *Game me kya hoga:*
1. HP bar 100/100 dikhega 
2. *ATTACK* dabayega to enemy HP ghat jayega
3. HP 0 hoga to *"BOOYAH! YOU WIN!"* aa jayega
4. *RESET* se dubara khel sakta hai
